# ColorSwitch
Developed a clone for the obstacle gaming app, Color Switch originally developed by Fortafy.

Working Demo of the game: https://drive.google.com/file/d/1spgBj5I_It9R3rSh0nB1dwJamaHPaqez/view?usp=sharing
